# H5补充认证项

## 机构跳转地址

/tongdun/apply/taobao

## 请求参数

| 参数 | 类型 | 是否必选 | 描述 |
| :--- | :---: | :--- | :--- |
| userName | String | 是 | 用户姓名 |
| userPhone | String | 是 | 用户手机号 |
| userIdCard | String | 是 | 用户身份证号 |
| returnUrl | String | 是 | 认证完成跳转页面 |
| verifyType | integer | 是 | 认证类型 |
