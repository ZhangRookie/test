# 订单还款展期H5

## 1.机构跳转地址

http://ybqb.zetafin.cn/index-ug

## 2.请求参数

| 参数 | 类型 | 是否必选 | 描述 |
| :--- | :--- | :--- | :--- |
| orderNo | string | 是 | 借款订单唯一编号 |
| sign | string | 是 | 签名:订单号 |

## 3.请求示例

http://ybqb.zetafin.cn/index-ug？orderNo=170808000016&sign=ef6852278c1445930290c127b0454965
